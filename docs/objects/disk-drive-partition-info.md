# Disk Drive Partition Info

## Object Definition


| Property | Description | Data Type |
| --- | --- | --- |
| name | The name of the disk drive partition represented by this object. Could be empty if the name could not be retrieved.| String |
| size_in_bytes | A non-negative number representing the size in bytes of the partition. A value of **-1** or **-1.0** means that the value could not be retrieved. | Float |
