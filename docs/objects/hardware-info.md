# Hardware Info

## Object Definition


| Property | Description | Data Type |
| --- | --- | --- |
| name | The name of the hardware device represented by this object. Could be empty if the name could not be retrieved.| String |
| vendor | The name of the manufacturer of the hardware device represented by this object. Could be empty if the value could not be retrieved. | String |
| type | A string (one of **PCI** or **USB** or **UNKNOWN**) representing the type of device. | String |
